## 🛠️ Tech Stack
- **Language:** Java 17
- **UI:** Selenium 4 + Grid
- **API:** REST Assured (TDD)
- **DB:** JDBC + H2
- **AI:** Ollama (Local LLM)
- **Reports:** Extent + Allure
- **CI/CD:** Jenkins
- **Containers:** Docker + Kubernetes

## 🚀 Quick Start
```bash
git clone https://github.com/YOUR_USERNAME/NexusQA.git
cd NexusQA
cp src/main/resources/config.properties.template \
   src/main/resources/config.properties
mvn test
```

- Extent Report: `reports/NexusQA_Latest_Report.html`
